LECTURES

강의 소개

현명역학원의 강의는 명리학의 기본 개념을 정확히 세우고, 실제 상담에서 활용할 수 있는 해석력까지 기르는 것을 목표로 합니다.

PROGRAM

사주명리 첫걸음

사주 팔자의 구성과 명리학을 공부할 때 반드시 잡아야 할 기본 관점을 정리합니다.

  • 음양오행의 기본 원리
  • 천간과 지지의 구조
  • 사주 원국을 읽는 첫 기준

CURRICULUM

커리큘럼

단순 암기보다 원리를 이해하고, 실제 사례에 적용할 수 있는 순서로 학습합니다.

  1. 명리 기초 음양오행, 천간지지, 십성, 용신의 기본 개념
  2. 원국 분석 일간 중심 해석, 격국, 조후, 신강신약 판단
  3. 운의 흐름 대운과 세운, 시기별 변화, 상담 포인트 정리
  4. 실전 상담 직업, 재물, 관계, 건강 질문에 맞춘 통변 연습

DIRECTOR

원장 소개

현명역학원은 상담 현장에서 자주 만나는 실제 질문을 바탕으로, 이론이 상담 언어로 바뀌는 과정을 차분하게 안내합니다. 단편적인 해석보다 전체 흐름과 맥락을 읽는 힘을 기르는 수업을 지향합니다.

교육 방향 원리 중심의 명리학 학습 실전 상담에 맞춘 통변 훈련 출간 도서와 연계한 체계적 복습
